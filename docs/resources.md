### GitHub 
GitHub是全球知名的基于Web的代码托管和协作开发平台，由Chris Wanstrath等人在2008年创立，后于2018年被微软收购。
它以Git分布式版本控制系统为基础，支持开发者对代码进行版本管理，提供公共和私有仓库服务。借助拉取请求、问题跟踪等功能，极大便利了团队协作开发。
